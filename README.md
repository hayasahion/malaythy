# malaythy
ملاذي منصة رقمية مبتكرة توفر للأطفال بيئة آمنة للتعبير عن مشاعرهم بطريقة غير مباشرة من خلال إنشاء قصص خيالية تفاعلية مدعومة بالذكاء الاصطناعي. تساعد المنصة على تعزيز الصحة النفسية، وتقديم مؤشرات استرشادية لأولياء الأمور والمرشدين لدعم الاكتشاف المبكر للمشكلات النفسية والسلوكية وتعزيز التواصل مع الطفل.
